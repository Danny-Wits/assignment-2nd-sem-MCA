# JavaScript Questions

1. **Sum & Average of Marks:** Enter the marks of students in 4 subjects and find out the sum and average of the marks obtained.
2. **Sum of Digits:** Enter a three digit number and find out the sum of all the digits.
3. **Prime Number Check:** Enter the number and find if the number is prime or not.
4. **Armstrong Numbers:** Write a program to print all the Armstrong numbers from 100 to 500.
5. **First 20 Primes:** Write a program to print first 20 prime numbers.
6. **Keyboard & Mouse Events:** Write a JavaScript Program to demonstrate all the events of the Keyboard and Mouse.
7. **Largest of Three (ReactJS):** Write a ReactJS Component to find the largest among three numbers.
8. **NodeJS Server Client Form:** Write a NodeJS Code to create a server and send the Rno, Name and Marks in 4 subjects from the client form to the server and return the sum of the marks obtained by the student.
